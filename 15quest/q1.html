Mensagem de Boas-vindas

Um script que exibe uma mensagem de boas-vindas ao usuário assim que a página carrega, utilizando uma caixa de diálogo informativa (alert). 

// Executa quando a página termina de carregar
window.addEventListener('load', function() {
  alert('Bem-vindo(a) ao site! 🎉');
});

▶ Executar

O alert é exibido automaticamente ao abrir a página.
