Corrigir Erro de Sintaxe

Corrija o erro de sintaxe no código abaixo para que a mensagem apareça corretamente no navegador. 

// ❌ Com erro
Console.Log( 'Olá Mundo' ).
↓ corrigido 
// ✅ Correto
console.log('Olá Mundo');
Erros encontrados:
1. Console → console (minúsculo)
2. Log → log (minúsculo)
3. ). → ); (ponto e vírgula)

Q 15

PBL — Sistema de Log de Acesso

PBL · Situação de Aprendizagem

Startup contratou um desenvolvedor para criar um sistema de log básico. Ao carregar: status do banco aparece no console, alert de 'Acesso Permitido' é exibido, e o nome do usuário logado é impresso no corpo da página.

Papel: Desenvolvedor Front-end Junior
Critérios: uso correto dos três comandos de saída JS
Tempo: 30 minutos 
// 1. Status do banco → console
console.log('Banco de dados: CONECTADO');

// 2. Acesso → alert
alert('Acesso Permitido');

// 3. Nome do usuário → document.write / DOM
document.write('Usuário: Ana Silva');
