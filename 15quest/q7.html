Corrigir Erro de Sintaxe

Corrija o erro de sintaxe no código abaixo para que a mensagem apareça corretamente no navegador. 

// ❌ Código com erro
Console.Log( 'Olá Mundo' ).
↓ corrigido 
// ✅ Código correto
console.log('Olá Mundo');

▶ Executar código correto

Clique em executar…

Erros encontrados:
1. Console → deve ser console (letras minúsculas)
2. Log → deve ser log (letras minúsculas)
3. ). → deve ser ); (ponto e vírgula no final)
