Sistema de Log de Acesso

Startup contratou um desenvolvedor para criar um sistema de log básico. Ao carregar o sistema, o status do banco de dados aparece no console, uma mensagem de 'Acesso Permitido' é exibida em um alert, e o nome do usuário logado é impresso no corpo da página.

Papel: Desenvolvedor Front-end Junior
Critérios: uso correto dos três comandos de saída JS
Tempo: 30 minutos 
// 1. Status do banco → console
console.log('Banco de dados: CONECTADO');

// 2. Acesso → alert
alert('Acesso Permitido');

// 3. Nome do usuário → document.write / DOM
document.write('Usuário: Ana Silva');
